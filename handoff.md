# Handoff: 山东众信官网项目

## 项目背景

本项目是为“山东众信价格评估拍卖有限公司”制作的专业服务品牌官网。网站最初参考了 `http://www.pccpv.com.cn/index.html` 的高级简洁、深蓝金色、机构官网风格，但不是复刻，而是重新组织为“众信专业服务体系”的官网。

当前内容口径不是单一公司孤立介绍，而是以“山东众信价格评估拍卖有限公司”为主，同时吸收 `ziliao/` 里的“山东众信土地房地产评估测绘有限公司”资料，形成价格评估、拍卖服务、房地产土地评估测绘、资产评估与司法辅助等综合服务形象。

## 网站结构

项目根目录：`professional-brand-site/`

- `index.html`：首页。包含首屏、服务领域、核心数据、专业能力、案例摘要、联系 CTA。
- `about.html`：关于我们。介绍价格评估拍卖公司和房地产评估测绘公司的协同关系、资质能力和发展节点。
- `services.html`：服务领域。分为价格评估、拍卖服务、房地产土地评估测绘、资产评估与司法辅助。
- `insights.html`：案例与动态。整理了资料中的真实案例，如济泰高速、京台高速、泰安环山路东延、山东科大泰山科技学院征迁、新泰市植物园等。
- `contact.html`：联系我们。展示电话、两个业务邮箱、地址、表单和地图占位。
- `styles.css`：全站样式。
- `script.js`：移动端导航展开、联系表单校验、首页动态栏目和案例列表接口渲染。
- `admin.html` / `admin.js`：内容管理后台，支持登录、文章管理、发布/下架、删除、上传封面和修改密码。
- `article.html`：站内文章详情页。
- `cms_server.py`：轻量 CMS 服务端，提供静态文件、SQLite 数据库、登录会话和内容 API。
- `data/`：运行后生成 SQLite 数据库，已加入 `.gitignore`。
- `uploads/`：后台上传图片目录，已加入 `.gitignore`。
- `assets/`：图片与 logo 资源。
- `ziliao/`：原始资料文档，不要删除。

## 设计风格

整体方向：

- 高级简洁、专业机构官网感。
- 主色为深蓝，辅助色为金色，背景以白色和浅灰为主。
- 页面结构规整，避免花哨动效。
- 首页首屏用商务/城市建筑大图，强调可信、专业、稳重。
- 服务模块使用图片卡片，参考原目标站的“业务入口块”，但样式更现代。

## 技术方案

当前保留静态 HTML/CSS/JS 结构，同时新增轻量 CMS 服务端。直接打开 HTML 仍可预览基础页面；如需动态公告、案例列表和后台管理，需要运行 `cms_server.py`。

本地预览：

```powershell
cd "D:\My Project\claude test\professional-brand-site"
python -m http.server 8080
```

然后访问：

```text
http://localhost:8080
```

也可以直接打开 `index.html` 预览。

内容后台预览：

```powershell
cd "D:\My Project\claude test\professional-brand-site"
python cms_server.py
```

然后访问：

```text
http://127.0.0.1:8080/
http://127.0.0.1:8080/admin
```

默认后台账号为 `admin` / `admin123`。正式部署前务必修改密码，或首次启动前设置 `CMS_ADMIN_PASSWORD` 环境变量。

修改约定：

- 编辑 HTML/CSS/JS 即可。
- 不需要安装依赖。
- 中文文件必须保持 UTF-8，之前曾出现过乱码，后续修改要注意编辑器编码。
- 如需改联系方式，请全站搜索旧值后替换，页脚在 5 个页面中各有一份。
