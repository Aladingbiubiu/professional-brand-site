# 山东众信价格评估拍卖有限公司官网

一个高级简洁风格的专业服务品牌官网静态版本，包含：首页、关于我们、服务领域、案例与动态、联系我们。

当前内容口径为“众信专业服务体系”，围绕价格评估、拍卖服务、房地产土地评估测绘、资产评估与司法辅助等业务展开。

## 本地预览

直接打开 `index.html` 即可浏览。若希望使用本地服务器：

```powershell
python -m http.server 8080
```

然后访问 `http://localhost:8080`。

## 内容后台

站点已加入轻量内容后台，使用 Python 标准库和 SQLite，不需要安装依赖。

```powershell
python cms_server.py
```

访问：

- 官网：`http://127.0.0.1:8080/`
- 后台：`http://127.0.0.1:8080/admin`

默认管理员账号：`admin`

默认密码：`admin123`

首次正式部署前请登录后台修改密码，或在首次启动前设置环境变量 `CMS_ADMIN_PASSWORD`。
